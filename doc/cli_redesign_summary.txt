CLI REDESIGN SUMMARY
====================

NEW BEHAVIOR:
-------------

1. Default (no options):
   ./memmap_analyzer.py memmap.txt
   → Shows ALL reports (table, stats, grouped, ascii, security)

2. Explicit --report:
   ./memmap_analyzer.py memmap.txt --report
   → Shows ALL reports (same as default)

3. Individual report options (show ONLY specified reports):
   --table      → Table view only
   --stats      → Statistics only
   --grouped    → Grouped by binary only
   --segments   → Segment overview only
   --ascii      → ASCII layout only
   --security   → Security analysis only

4. Crash analysis options (show ONLY crash context):
   --pc <addr>  → Crash analysis for PC
   --lr <addr>  → Crash analysis for LR
   --sp <addr>  → Crash analysis for SP
   --fp <addr>  → Crash analysis for FP

5. Multiple options combined:
   ./memmap_analyzer.py memmap.txt --table --stats
   → Shows table AND stats only (not all reports)

6. Invalid options:
   ./memmap_analyzer.py memmap.txt --invalid
   → Error message + suggests --help (does NOT auto-show help menu)

KEY CHANGES:
------------
- Added --report option for explicit "all reports" request
- Each option now shows ONLY its specific report
- No more dumping all data when specific options are given
- Error handling improved (no auto-help on invalid options)
- Multiple options can be combined to show specific reports
